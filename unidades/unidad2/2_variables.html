Variables

Tip

En Ruby, todo es un objeto. Las variables en Ruby son referencias a objetos. Por lo tanto vamos a poder acceder a los métodos para extraer información de los objetos.

Para declarar una variable en Ruby, solo necesitas escribir el nombre de la variable seguido de un signo igual y el valor de la variable.

nombre = "Juan"
edad = 25

Para imprimir el valor de una variable en Ruby, puedes utilizar el método puts.

puts nombre
puts edad

Convenciones para declarar variables

Ruby al igual que otros lenguajes de programación tiene convenciones para declarar variables. Las convenciones son reglas que se utilizan para nombrar variables, métodos, clases, etc. de manera consistente. Las convenciones ayudan a que el código sea más fácil de leer y entender.

Para declarar variables por convesnsión se utiliza snake_case. Es decir que las variables se declaran en minúsculas y si tienen más de una palabra se separan con guiones bajos.

Ejemplo:

nombre_completo = "Juan Perez"

Las constantes se declaran en mayúsculas y si tienen más de una palabra se separan con guiones bajos.

Ejemplo:

PI = 3.14159

Las variables de instancia se declaran con un signo de arroba al principio.

Ejemplo:

@nombre = "Juan"

Las variables de clase se declaran con dos signos de arroba al principio.

Ejemplo:

@@nombre = "Juan"

Las variables globales se declaran con un signo de dólar al principio.

Ejemplo:

$nombre = "Juan"

En Ruby, las variables locales se declaran en minúsculas.

Ejemplo:

nombre = "Juan"

Las variables de bloque se declaran con un signo de ampersand al principio.

Ejemplo:

&nombre = "Juan"

Tipo de variables

En Ruby, las variables pueden ser de diferentes tipos. Algunos de los tipos de variables más comunes en Ruby son:

  • Integer: Un número entero.
  • Float: Un número decimal.
  • Boolean: Un valor booleano (true o false).
  • String: Una secuencia de caracteres.
  • Array: Una colección de elementos.
  • Hash: Una colección de pares clave-valor.
  • Nil: Un valor especial que representa la ausencia de un valor.
  • Symbol: Un objeto inmutable que representa un nombre o un identificador.
  • Range: Un objeto que representa un rango de valores.

Ahora vamos a analizar los diferentes tipos de variables en Ruby.

Intergers

En Ruby, los integers son números enteros. Puedes realizar operaciones matemáticas con integers en Ruby.

numero = 5
puts numero

Floats

En Ruby, los floats son números decimales. Puedes realizar operaciones matemáticas con floats en Ruby.

numero = 5.5
puts numero
Tip

Ruby tiene algo diferente a otros lenguajes de programación, por ejemplo para trabajar con valores de números muy grandes, en otros lenguajes de programación se necesita trabajar con librerías especiales, en Ruby no es necesario.

Ejemplo:

x = 2**1000
puts x
x.class

Como podemos observar en el ejemplo anterior, el resultado de la operación es un número muy grande y no es necesario trabajar con librerías especiales.

Strings

En Ruby, los strings son objetos que representan una secuencia de caracteres. Puedes crear strings en Ruby utilizando comillas simples o comillas dobles.

nombre = "Juan"
apellido = 'Perez'

Sin embargo tabien existen otras 2 formas de declarar strings en Ruby:

Tip

Estas caracteristicas pueden ser importantes cuando necesitas utilizar comillas dobles o simples dentro de un string.

nombre = %q(Juan)
apellido = %Q(Perez)

Otra caracteristica importante es la interpolación de strings:

nombre = "Juan"
apellido = "Perez"
nombre_completo = "#{nombre} #{apellido}"
puts nombre_completo
Tip

Esta caracteristica puede ser útil cuando necesitas concatenar strings, en este caso Ruby interpreta que el contenido entre las llaves es una variable y necesito remplazarlo en el string.

Warning

Es importante tener en cuenta que si utilizas comillas simples en la interpolación de strings, Ruby no interpretará el contenido entre las llaves como una variable.

Ejemplo:

nombre = 'Juan'
apellido = 'Perez'
nombre_completo = '#{nombre} #{apellido}'
puts nombre_completo

En este caso el resultado será #{nombre} #{apellido}. Esto se debe a que Ruby interpreta el contenido entre las llaves como un string y no como una variable.

Métodos para trabajar con strings

Para poder trabajar con strings en Ruby, existen diferentes métodos que nos permiten realizar operaciones con strings.

Para comprobar que el tipo de dato que estamos utilizando es un string, podemos utilizar el método .class.

nombre = "Juan"
puts nombre.class

De la misma forma para conocer que métodos podemos utilizar con un string, podemos utilizar el método .methods.

nombre = "Juan"
puts nombre.methods
Tip

Los 10 métodos más comunes para trabajar con strings en Ruby son:

  • .length: Devuelve la longitud de un string.
  • .upcase: Convierte un string a mayúsculas.
  • .downcase: Convierte un string a minúsculas.
  • .capitalize: Convierte la primera letra de un string a mayúsculas.
  • .reverse: Invierte un string.
  • .split: Divide un string en un array de strings.
  • .strip: Elimina los espacios en blanco al principio y al final de un string.
  • .gsub: Reemplaza una subcadena de un string por otra subcadena.
  • .include?: Verifica si un string contiene una subcadena.

Ejemplo:

5puts nombre.reverse

Warning

Es importante tener en cuenta que los strings en Ruby son inmutables. Esto significa que una vez que se crea un string, no se puede modificar. Si necesitas modificar un string, puedes crear un nuevo string con el resultado de la operación.

Ejemplo:

nombre = "Juan"
nombre.upcase
puts nombre

En este caso el resultado será Juan. Esto se debe a que el método .upcase no modifica el string original, sino que crea un nuevo string con el resultado de la operación.

Tip

El método .gsub es muy útil para reemplazar una subcadena de un string por otra subcadena. Puedes utilizar el método .gsub para reemplazar todas las ocurrencias de una subcadena en un string. Sin embargo no crea una nueva cadena, sino que modifica la cadena original.

Ejemplo:

nombre = "Juan"
nombre2 = nombre.gsub("Juan", "Pedro")
puts nombre
puts nombre2

En este caso el resultado será Juan y Pedro. Esto se debe a que el método .gsub modifica el string original y crea un nuevo string con el resultado de la operación.

Concatenación de strings

Para concatenar strings en Ruby, puedes utilizar el operador +.

nombre_completo = nombre + " " + apellido
puts nombre_completo

También puedes utilizar el método .concat para concatenar strings.

nombre_completo = nombre.concat(" ").concat(apellido)
puts nombre_completo

Tambien podemos utilizar el operado * para repetir un string.

nombre_completo = nombre * 3
puts nombre_completo

Interpolación de strings

Para interpolar strings en Ruby, puedes utilizar el operador #{}.

nombre_completo = "#{nombre} #{apellido}"
puts nombre_completo

edad = 25
puts "Hola, mi nombre es #{nombre} #{apellido} y tengo #{edad} años"

:::.{.callout-tip .justify} Cada vez que nosotros declaramos una variable, Ruby se encarga de asignar un identificador único a esa variable. Por lo tanto, si nosotros declaramos una variable con el mismo nombre en diferentes partes de nuestro código, Pero con diferente valor Ruby no va a tener problemas en identificar a que variable nos estamos refiriendo.

Ejemplo:

puts nombre.object_id

Ahora, que sucede cuando declaramos el mismo valor en diferentes variables, Ruby va a asignar el mismo identificador a esas variables.

Ejemplo:

nombre = "Juan"
nombre2 = "Juan"
puts nombre.object_id
puts nombre2.object_id

Como podemos observar en el ejemplo anterior, declaramos dos variables con el mismo valor, Ruby asigna identificadores únicos a cada variable. Por lo tanto podemos concluir que Ruby asigna un identificador único a cada variable, independientemente del valor que tenga. :::

Symbols

En Ruby, los symbols son objetos inmutables que representan un nombre o una identificador. Puedes crear symbols en Ruby utilizando el prefijo :.

nombre = :juan
apellido = :perez
Tip

Los symbols son muy útiles en Ruby porque son inmutables y ocupan menos memoria que los strings. Los symbols son muy útiles en Ruby porque son inmutables y ocupan menos memoria que los strings.

Ejemplo:

nombre = :juan

puts nombre2.object_id

Como podemos observar en el ejemplo anterior, declaramos dos variables con el mismo valor, Ruby asigna el mismo identificador a cada variable. Por lo tanto podemos concluir que Ruby asigna el mismo identificador a cada variable, independientemente del valor que tenga.

Arrays

En Ruby, los arrays son objetos que representan una colección de elementos. Puedes crear arrays en Ruby utilizando corchetes.

nombres = ["Juan", "Maria", "Pedro"]
edades = [25, 30, 35]
Tip

Para poder saber si realmente el tipo de dato que estamos utilizando es un array, podemos utilizar el método .class. Y para conocer cuales son los métodos que podemos utilizar con un array, podemos utilizar el método .methods.

Ejemplo:

nombres = ["Juan", "Maria", "Pedro"]
puts nombres.class
puts nombres.methods

Como podemos observar en el ejemplo anterior, se utilizan los métodos .class y .methods para trabajar con arrays en Ruby.

En Ruby, todo es un objeto, es por ello que el método .methods nos devuelve una lista de los métodos disponibles para un array.

Ejemplo:

nombres = ["Juan", "Maria", "Pedro"]
puts nombres.methods
puts nombre.methods.class
puts nombre.methods.class.size

En el ejercicio anterior se utilizan los métodos .methods y .class, el resultado de puts nombre.methods.class será Array. Esto se debe a que el método .class nos devuelve la clase de un objeto, en este caso un array.

Tambien podemos observar que la salida de puts nombre.methods.class.size será 180. Esto se debe a que el método .methods nos devuelve una lista de los métodos disponibles para un array, y el método .size nos devuelve la cantidad de métodos disponibles para un array.

Para acceder a un elemento de un array en Ruby, puedes utilizar el índice del elemento.

puts nombres[0]
puts edades[1]
Tip

Algo especial de los arrays en Ruby es que el índice de los elementos comienza en 0. Por lo tanto, el primer elemento de un array tiene un índice de 0, el segundo elemento tiene un índice de 1, y así sucesivamente. Pero tambien podemos utilizar índices negativos para acceder a los elementos de un array.

Ejemplo:

nombres = ["Juan", "Maria", "Pedro"]
puts nombres[-1]
puts nombres[-2]

En este caso el resultado será Pedro y Maria. Esto se debe a que el índice negativo nos permite acceder a los elementos de un array desde el final del array.

¿Qué sucede si ingresamos un valor que está fuera de la capacidad de nuestro array?

Ejemplo:

nombres = ["Juan", "Maria", "Pedro"]
puts nombres[100]

En este caso el resultado será nil. Esto se debe a que el índice que ingresamos está fuera de la capacidad de nuestro array, por lo tanto Ruby nos devuelve nil.

Warning

Es importante tener en cuenta que los arrays en Ruby son mutables. Esto significa que puedes modificar un array después de crearlo. Si necesitas modificar un array, puedes utilizar los métodos disponibles para los arrays en Ruby.

Ejemplo:

nombres = ["Juan", "Maria", "Pedro"]
nombres.push("Luis")
puts nombres

En este caso el resultado será Juan, Maria, Pedro, Luis. Esto se debe a que el método .push añade un elemento al final de un array.

:::{.callout-tip .justify} Otro dato importante de los arrays en Ruby es que no tienen un tipo de dato fijo, es decir que podemos almacenar diferentes tipos de datos en un array.

Ejemplo:

datos = ["Juan", 25, 1.75, ["Pedro", 35, 1.80]]
puts datos

En este caso el resultado será Juan, 25, 1.75, Pedro, 35, 1.80. Esto se debe a que podemos almacenar diferentes tipos de datos en un array, inclusive otro array.

Hashes

En Ruby, los hashes son objetos que representan una colección de pares clave-valor. Puedes crear hashes en Ruby utilizando llaves.

persona = {nombre: "Juan", edad: 25}

Para acceder a un valor de un hash en Ruby, puedes utilizar la clave del valor.

puts persona[:nombre]
puts persona[:edad]

Para añadir un par clave-valor a un hash en Ruby, puedes utilizar la clave y el valor.

persona[:apellido] = "Perez"
puts persona

Para eliminar un par clave-valor de un hash en Ruby, puedes utilizar el método .delete.

persona.delete(:apellido)
puts persona

Booleans

En Ruby, los booleans son valores booleanos (true o false). Puedes utilizar los operadores lógicos para comparar valores booleanos en Ruby.

verdadero = true
falso = false

Nil

En Ruby, nil es un valor especial que representa la ausencia de un valor. Puedes utilizar nil para inicializar una variable sin asignarle un valor.

valor = nil
puts valor

Ranges

En Ruby, los ranges son objetos que representan un rango de valores. Puedes crear ranges en Ruby utilizando el operador .. o el operador ….

rango1 = 1..5
rango2 = 1...5

Los ranges son muy útiles en Ruby porque puedes utilizarlos para iterar sobre una secuencia de valores.

rango = 1..5
rango.each do |numero|
  puts numero
end

En Ruby puedes utilizar los operadores aritméticos para realizar operaciones matemáticas.

2 + 3

2**3

Tip

Cuando realizamos una división entre dos integers, el resultado es un integer. Si queremos obtener un float como resultado, podemos incorporar un float en la operación.

Ejemplo:

2 / 3.0
